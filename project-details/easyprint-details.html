Project information

  • Name: Easyprint
  • Category: Fullstack Web App
  • Client: Millena Móveis
  • Industry: Retail (furniture store chain)
  • Project date: 01 March, 2020

Avoiding fines with a simple price tag printing management solution

Easyprint is a system that allows stores to transform big chunks of data into a PDF containing price tags with the information uploaded. The client is a furniture store chain from the northeast of Brazil present in around 40 locations in the region.

My client needed a way to create their price tags layout files so each store could print the right prices and information about the products. Before Preço Fácil they used to send Excel sheets to each store via email and an employee should use it to print the layout.

This process led them to be prone to government fines up to R$ 50.000 (around U$ 10.000) because they had to always ensure that the data was synced between all the stores and also that there were no changes made in the sheet by someone in the store.

The system is still evolving and the plan is to transform other manual processes regarding pricing into new features inside the app.
